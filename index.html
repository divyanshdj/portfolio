HTML (APRIL 2023)
HTML (Hypertext Markup Language) is a standard markup language used to create and structure content on the web. It consists of tags and elements that define the structure and presentation of web pages, including text, images, links, and more. Browsers interpret HTML to display web content to users.
CSS (APRIL 2023)
CSS is a stylesheet language used for describing the visual presentation of web pages written in HTML. It defines how elements should be displayed, including layout, colors, fonts, and more.
JS (MAY 2023)
JavaScript is a versatile scripting language used to add interactivity and dynamic behavior to web pages. It allows for client-side scripting and can manipulate web content and respond to user actions.
SQL (MAY 2023)
SQL is a programming language designed for managing and querying relational databases. It's used to create, retrieve, update, and delete data from databases and is essential for data-driven web applications and systems.
